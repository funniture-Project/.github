## 📋개요
이 프로젝트는 가구 및 가전 제품의 렌탈 서비스를 제공하는 **Funniture** 플랫폼입니다.  
단순 대여 서비스를 넘어 렌탈부터 반납까지 관리하여 사용자와 제공자 모두에게 편리한 **원스톱 관리 서비스**를 제공하고 있습니다.
